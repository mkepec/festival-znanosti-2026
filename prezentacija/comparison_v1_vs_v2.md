# Usporedba: Originalna (v1) vs. Updated prezentacija (v2)

## 🔥 Ključne razlike i poboljšanja

### **Struktura i fokus:**

| **V1 (Original)** | **V2 (Updated)** |
|-------------------|------------------|
| 11 slideova | 11 slideova (optimizirano) |
| Generička AI objašnjenja | Tesla-specific analogije |
| Tehnički fokus | Elektrotehnički fokus + praktičnost |
| Pasivna prezentacija | Interaktivna s pitanjima |

### **Najveće dodane vrijednosti u V2:**

## 🚀 **1. "Perfect Storm" koncept (Slide 5)**
**V1:** Jednostavan RAG workflow objašnjenje
**V2:** Objašnjava ZAŠTO je Tesla+AI moguće baš sada:
- Digitalizirani Tesla dokumenti
- Cloud computing snaga
- Transformer algoritmi

**Impact:** Učenici razumiju povijesni kontekst, ne samo tehniku

## ⚡ **2. Elektrotehničke analogije kroz sve**
**V1:** "LLM kao inteligentni transformator"
**V2:** Kompletni signal processing chain:
```
Tesla PDF → Multiplexer → A/D → Memory → Selector → Amplifier → Output
```

**Impact:** Povezuje novo znanje s postojećim elektrotehničkim konceptima

## 🎯 **3. "Voltage Drop in Prompts" (Slide 7)**
**V1:** Nema prompt quality objašnjenje
**V2:** Kvaliteta prompta = napon u krugu
- Nizak napon → loš output
- Visok napon → kvalitetan output

**Impact:** Praktične vještine za korištenje AI-ja

## 🔋 **4. Energy Accounting (Slide 9)**
**V1:** Kratko spominjanje energetske potrošnje
**V2:** Detaljne usporedbe:
- 1 Tesla coil demo = 83 AI odgovora
- Povezivanje s Tesla temom energije

**Impact:** Svjesnost o realnim troškovima AI tehnologije

## 🛠️ **5. Two-Track Workshop struktura**
**V1:** Jedan pristup (n8n)
**V2:** Dva pristupa:
- Track 1: Visual n8n workflow
- Track 2: Backup ChatGPT+docs

**Impact:** Backup plan za tehnička ograničenja
